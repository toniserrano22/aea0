5. Què és una arrow function? A quin ES apareix (quin any).

Son una nova forma de definir funcions i ha distintes variants de la sintaxis, elimina la palabra "function" i posa la fletxa. Apareix al ES6 l'any 2015


6. Què és un callback? Posa un exemple.

Es una funció que se pasa a otra funció com un argument, que despres es invoca dintre de la funció externa para completar algun tipus de rutina o acció
function saludar(nombre) {
alert('Hola ' + nombre);
}
function procesarEntradaUsuario(callback) {
var nombre = prompt('Por favor ingresa tu nombre.');
callback(nombre);
}
procesarEntradaUsuario(saludar);


7. Què és una promesa en Javascript (promise). Posa un exemple. Explica també la diferència entre la programació síncrona i la programació asíncrona, posa un exemple.

Una promesa es un objecte que representa la terminació o el fracas d'una operació asíncrona. Donat que la majoria de las personas consumen promises ja creades, aquesta guía explicara primer com comsumirles, y despres com crearles.
function exitoCallback(resultado) {
console.log("Archivo de audio disponible en la URL " + resultado);
}

function falloCallback(error) {
console.log("Error generando archivo de audio " + error);
}

crearArchivoAudioAsync(audioConfig, exitoCallback, falloCallback);
En la comunicació asincrònica, no hi ha coincidència temporal entre emissor i receptor. En canvi, en la comunicació sincrònica, la coincidència temporal ha d'existir perquè es produeixi la comunicació (es transmeti el missatge).
